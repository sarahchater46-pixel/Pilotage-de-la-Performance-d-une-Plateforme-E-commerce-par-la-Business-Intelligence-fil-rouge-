📊 E-commerce Business Intelligence Dashboard                                  
📌 Overview

This project presents an end-to-end Business Intelligence solution developed to analyze the performance of a Moroccan e-commerce store.

The solution integrates data from multiple sources, applies data preparation and transformation techniques, and delivers interactive Power BI dashboards to support data-driven decision-making.

The analysis focuses on three strategic business areas:                             

📦 Product Performance                                              
📣 Marketing Performance                                                              
🚚 Logistics Performance                                           
🎯 Project Objectives                                                                

The main objectives of this project are to:                                                             

Analyze sales performance and product profitability.                                                  
Monitor marketing campaign effectiveness.                                                  
Evaluate logistics and delivery operations.                                                  
Develop interactive dashboards for business users.
Demonstrate a complete Business Intelligence workflow.                                                  
📂 Data Sources                                                  

The project combines several data sources to simulate a real business environment.

📊 Kaggle — Main e-commerce dataset containing sales, orders, products and customer data.                                                  
🌍 Google Maps — Delivery company information collected through Web Scraping.                                                  
🗄️ SQL Database — Customer data simulating a CRM system.                                                  
🔗 FastAPI — Marketing data generated through a local REST API.                                                  
📄 CSV Files — Product, logistics and fact tables used in the analytical model.                                                  
🛠️ Data Preparation                                                  

Data preparation was carried out using Python before loading the data into Power BI.

The preprocessing phase included:

Exploratory Data Analysis (EDA)
Data Cleaning
Missing Values Handling
Data Validation
Feature Engineering
Marketing Data Generation
Data Transformation

The processed datasets were then integrated into Power BI using Power Query.

🏗️ Data Modeling

The analytical model follows a Star Schema architecture to ensure efficient reporting and optimized analytical performance.

The model consists of:                                                  

⭐ One Fact table containing sales transactions.                                                  
📑 Dimension tables for Customers, Products, Dates, Marketing and Logistics.                                                  

Relationships between tables were designed to support reliable DAX calculations and interactive reporting.

📈 Dashboards                                                  
📦 Product Dashboard                                                  

This dashboard provides insights into product and customer performance through key business indicators such as Gross Sales, Net Profit, Quantity Sold, Average Order Value and Sales Growth.

It helps identify top-performing products, customer behavior and overall commercial performance.

📣 Marketing Dashboard

This dashboard evaluates marketing performance using KPIs such as Customer Acquisition Cost (CAC), Click-Through Rate (CTR), Active Campaigns, Total Clicks and Total Impressions.

It also provides insights into marketing channels and campaign effectiveness.

🚚 Logistics Dashboard

The logistics dashboard monitors operational performance through delivery success rates, return analysis, carrier performance, payment methods and return reasons.

It helps evaluate the efficiency of logistics operations and customer service.

This project demonstrates how data from multiple heterogeneous sources can be integrated into a unified Business Intelligence solution.

By combining Python, SQL, FastAPI and Power BI, the solution provides interactive dashboards that enable decision-makers to monitor business performance, evaluate marketing effectiveness and optimize logistics operations through reliable, data-driven insights.
